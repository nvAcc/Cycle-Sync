# LUNA - AI Powered Menstrual Health Tracker

LUNA is an intelligent menstrual wellness platform designed for privacy and precision. Beyond predicting cycles, it features an empathetic AI chatbot for emotional support and a secure community for shared experiences. LUNA combines data accuracy with compassionate care to be your reliable companion for total menstrual health.

## Key Features

*   **Intelligent Cycle Tracking:** Accurate predictions for fertile windows and upcoming periods.
*   **Empathetic AI Companion:** A built-in chatbot offering emotional support and wellness advice.
*   **Privacy-First Architecture:** Sensitive health data is stored locally on your device, ensuring complete privacy.
*   **Secure Community:** An anonymous, supportive space to share experiences.
*   **Health Insights:** Interactive charts for understanding cycle patterns.

## Tech Stack

*   **Frontend:** React (Vite), Tailwind CSS, shadcn/ui
*   **Backend:** Node.js, Express.js, PostgreSQL
*   **AI/ML:** TensorFlow.js

## Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    cd Cycle-Sync
    ```

2.  Install dependencies:
    ```bash
    npm install
    ```

3.  Set up environment variables in a .env file:
    ```env
    DATABASE_URL=postgresql://...
    SESSION_SECRET=secret
    ```

4.  Start the application:
    ```bash
    npm run dev
    ```
